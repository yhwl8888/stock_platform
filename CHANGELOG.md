# 更新日志 (CHANGELOG)

## [v2.0.0] - 2026-05-28

### 🆕 新增功能

#### 5个新高胜率选股策略
- **多因子共振 (multi_factor_resonance)**: 结合趋势（MA5>MA20>MA60）+ 金叉 + 放量 + 动量，适用于趋势追踪场景
- **MACD+RSI+成交量 (macd_rsi_volume)**: MACD金叉 + RSI超卖 + 成交量放大，捕捉超卖后反弹机会
- **布林带RSI均值回归 (boll_rsi_mean_reversion)**: 跌破下轨+RSI超卖买入，触及上轨或RSI超买卖出，适合震荡行情
- **海龟交易法则 (turtle_trading)**: 经典趋势跟踪策略，突破20日高点买入，跌破10日低点卖出
- **RSI反转+成交量 (rsi_reversal_volume)**: RSI超卖反转+成交量暴增确认，捕捉超跌反弹机会

#### 前端优化
- 策略选择器改为动态加载，从 `/api/strategies` 接口获取最新策略列表
- 新增 `loadStrategies()` 函数自动填充所有可用策略

### 🔧 改进与优化

#### 参数优化引擎
- 大幅减少参数组合数量（从最多6561个减少到200个以内）
- 添加最大组合数限制（MAX_COMBINATIONS = 200）
- 超过限制时使用随机采样（固定种子42保证可复现）
- 优化前后对比：
  | 策略 | 之前 | 现在 |
  |------|------|------|
  | multi_factor_resonance | 2187 | 4 |
  | macd_rsi_volume | 729 | 32 |
  | boll_rsi_mean_reversion | 2187 | 64 |
  | turtle_trading | 256 | 16 |
  | rsi_reversal_volume | 6561 | 128 |

#### 数据获取模块 (fetcher.py)
- ✅ **修复北京交易所股票支持**: 移除8/4开头代码的限制
- ✅ **添加请求频率限制**: 最小间隔300ms，防止被API限流
- ✅ **增强错误处理**:
  - 添加 HTTP 状态码检查 (`resp.raise_for_status()`)
  - 添加 JSON 解析错误专门处理
  - 区分网络错误和其他异常类型
  - 完善日志输出便于调试
- ✅ **延长超时时间**: 从5秒延长到10-20秒
- ✅ **更新User-Agent**: 使用完整浏览器标识避免被识别为爬虫

#### 回测引擎 (backtest.py)
- 修复 volume 参数传递方式，通过 params 字典统一管理
- 所有策略函数可通过 `params.get("volume")` 访问成交量数据

#### 前端错误处理 (app.js)
- 回测和优化接口添加响应状态检查
- 先获取文本再尝试JSON解析
- 提供更清晰的错误信息提示

### 🎯 四维评分系统说明

评分维度（满分100分）:
- **趋势评分** (30分): 均线排列 + MACD状态 + 趋势一致性
- **超买超卖** (25分): KDJ指标 + RSI指标
- **量价分析** (25分): 成交量形态 + OBV资金流向
- **位置运动** (20分): 布林带位置 + 支撑阻力位

评分标准:
- ≥75分: 强买入信号
- 60-74分: 弱买入信号
- 40-59分: 观望
- 25-39分: 弱卖出信号
- <25分: 强卖出信号
